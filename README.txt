Bienvenid@ al aplicativo: Gestión de clientes

El aplicativo registra los datos personales de los clientes, el propósito de la aplicación es, que el usuario pueda manejar la base de datos, para una mejor toma de decisiones con la información obtenida.  

Instrucciones de uso:
Una vez ejecutado el aplicativo, brinda las siguientes opciones:

1. Agregar cliente
2. Listar clientes
3. Actualizar cliente
4. Eliminar cliente
5. Buscar clientes por ciudad
0. Salir


Descripción de cada opción:

1:  Agregar cliente: Guarda información sobre el nuevo cliente,  incluyendo: nombre, primer apellido, segundo apellido, sexo, ciudad, fecha de nacimiento, teléfono, correo.

2:  Listar clientes: Permite visualizar todos los clientes en la base de datos.

3:  Actualizar cliente: Permite modificar la información de un cliente.

4: Eliminar un cliente: Elimina al cliente de la base de datos.

5:  Buscar cliente por ciudad: Permite buscar clientes por ciudad, ver la lista de los clientes que son de esta ciudad.

0. Salir: Permite salir del aplicativo.
